# Hola, soy José Israel Manrique 🚀

Soy un Desarrollador Junior Full Stack de aplicaciones web y móviles, profundamente enfocado en construir experiencias digitales que destaquen por ser vanguardistas, altamente funcionales y con un diseño visual impecable.

Disfruto combinar el ingenio con la tecnología y un estricto rigor por los detalles. Mi objetivo principal es transformar conceptos e ideas abstractas en soluciones tecnológicas robustas, intuitivas y orientadas al usuario final. Día a día, me dedico a potenciar mis capacidades en el desarrollo frontend y backend, perfeccionando el diseño responsivo, la integración de sistemas complejos y la aplicación de las mejores prácticas de la industria del software.

## 💻 Lo que hago

* Desarrollo de sitios web responsivos
* Maquetación de interfaces modernas
* Creación de landing pages y páginas institucionales
* Integración de formularios y funcionalidades web
* Desarrollo frontend con enfoque en experiencia de usuario
* Proyectos full stack en constante aprendizaje

## 🧰🛠️ Tecnologías y herramientas

HTML · CSS · JavaScript · TypeScript · React · Next.js · Tailwind CSS · Python · FastAPI · Git · GitHub · Vercel · Figma · Jira · 

## 🧠 Actualmente aprendiendo

Me encuentro inmerso en el estudio profundo de la arquitectura Full Stack y la consolidación de las mejores prácticas de codificación. Además, estoy ampliando y puliendo mis conocimientos técnicos en accesibilidad web, optimización de bases de datos, mejora del rendimiento general de las aplicaciones (performance) y estrategias eficientes de despliegue.

## 💼 Sobre mi forma de trabajar

Me considero un profesional creativo, metódico, analítico, estructurado y con un gran sentido de la resolución de problemas. Disfruto desarrollar proyectos que tengan un propósito claro desde su concepción, asegurándome de mantener un estándar de calidad altísimo tanto en el diseño visual como en la lógica interna del código. Considero que cada nuevo desafío es una oportunidad para aprender algo nuevo y perfeccionar mi técnica.
